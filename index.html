Project Racegame

The very first project I had the pleasure to work on.
This project involved making a game using c# code. The game needed to meet the following requirements:

  • 2 players have to be able to race each other.
  • Both players have to be able to see how many laps they have driven on the circuit.
  • Both players have to have a limited supply of fuel and hit points.
  • When the fuel or hitpoints reach zero, the player needs to visit a pitstop to repair/refuel their car.
  • Both players need to have a speedometer.

For this project my project group and I decided to base it on pizza delivery.
There are 2 pizza delivery men who are competing with each other to see which of them can deliver the pizza to the customer first

Project Home Automation

The second project I had the pleasure to work on.
This project involved making use of apps and an Arduino unit. The Project needed to meet the following requirements:

  • A phone with the app built on it needs to be able to connect with the Arduino unit.
  • Through the app the user should be able to turn something off/on in their home.

For this project my project group and I got the assignment to first build an app that would interface with the Arduino unit to turn lights on/off.
The second assignment was to make something like the first assignment but the idea for it was left to the project group, my group decided on an app that would involve a security system for the user's front door. This app was able to make use of a security camera installed at the user's front door that would show a live feed on the app. The app was built this way to make it safer for someone to open their front door.
The final assignment was to build an app that would be able to both open and close a small bridge, it had to be able to get the signals to open and or close through either the app or through another bridge placed next to it. The signal from said other bridge would be given by making use of a photoconductive cell that would give signals when the light it sees gets interrupted by a foreign object.

Project Website

The third project I had the pleasure to work on.
This project involved making a website for a customer. The website needed to meet the following requirements:

  • The user needs to be able to log in
  • The website needs to have an Admin that has to agree to new users applying for the website.
  • Only people that are logged in and approved by the Admin are capable of altering data on the site.
  • People that are not logged in or approved are only capable of browsing the content and printing the data on it.

For this project my project group and I got the assignment to work on a website that was capable of making planners for classrooms within the school. On this website the approved people of the admin would be able to log in on the website and add new classrooms to the database or be able to change data in one of the planners for a existing classroom.
Every person that visited the website was capable of viewing the planners and/or printing them.
After a month the old data for the classrooms would be reset to keep the data stored to a minimum.

Project App

The fourth and final project of the first year within the course.
This project involved building an app for one of the teachers of the school.
Our assignment was to build a Routeplanner for use within the school.
The app needed to meet the following requirements:

  • The user has to be able to choose a specific classroom out of a given amount of classrooms
  • The user has to be able to see a map with a given route from his/her location to the chosen classroom

For this project our app made use of the Dijkstra method to find the shortest route within a given amount of locations. This was used so that the user would be able to get to their classroom as quickly as possible.
This app also made use of an easy to use NumberPicker interface with which one could pick a specific department within the school, and then be able to only choose from classrooms within said department.
